Take a look at this partial implementation of a system with support for byte-sized memory accesses. The attacker and the victim program do not share memory, but they do use the same direct-mapped cache which has a size of 32 bytes and a block size of 1 byte. The victim program leaks the secret FLAG via a cache side channel. Your task is to determine the value of FLAG by performing memory accesses and running the victim program. Use your browser console to make sys.attackerAccess() and sys.victimProgram() function calls and capture the flag! For information on caches, see here.

Notice that a cache miss takes 3 seconds to retrieve the block from main memory, but each subsequent access of the same block is immediate since it is a cache hit.
